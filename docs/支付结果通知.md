# 支付结果通知 YQN02030

## 1. 使用说明

> [!IMPORTANT]
> 直联通知为辅助功能，不能完全依赖通知功能进行业务处理。需要同步开发业务查询接口，如果长时间没有收到通知，请通过相应的查询接口来获取业务状态。
> 
> 收到业务完成通知，不代表收方都能入账成功，到账以收方渠道为准。

请先解析通知内容中的 `msgtyp` 字段的值获取通知子类型。各子类型如下：

*   **FINS**: 业务完成通知
*   **FINB**: 支付退票通知

`msgdat` 为具体通知内容。

---
