# 账务查询
- 更新时间：2026-01-19 14:02:38
- 版本号：1.0.0

账务查询主要用于查询本单位账户信息或集团公司查询全国范围内在招商银行开户的子公司账户信息，包括查询账户详细信息、账户交易信息、账户实时余额、账户历史余额以及电子回单信息。

## 17. 单笔回单查询DCSIGREC
### 使用场景
该接口主要用于根据流水号查询单笔交易回单。日期需为回单产生的日期，流水号可通过账户交易信息查询trsQryByBreakPoint请求返回接口transSequenceIdn字段获取。

### 使用说明
可返回支持自助打印的回单

### 请求报文
| 字段名称 | 字段ID | 数据类型 | 是否必输 | 描述 |
| ---- | ---- | ---- | ---- | ---- |
| 账号 | eacnbr | String(35) | Y |  |
| 查询日期 | quedat | String(10) | Y | yyyy-MM-dd格式 |
| 交易流水号 | trsseq | String(15) | Y |  |
| 打印模式 | primod | String(3) |  | 空时默认PDF，可选PDF或者OFDEX |

### 响应报文
| 字段名称 | 字段ID | 数据类型 | 是否必输 | 描述 |
| ---- | ---- | ---- | ---- | ---- |
| 验证码 | checod | String |  |  |
| 单笔回单返回的数据流 | fildat | String |  | 返回Base64后的数据，Base64解码后写入文件可得到PDF或OFD文件 |
| 回单实例号 | istnbr | String |  |  |
