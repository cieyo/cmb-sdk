## 2.3.1 账务变动通知（YQN01010）
### 重要说明
1. 直联通知为辅助功能，不能完全依赖通知功能进行业务处理，需同步开发业务查询接口；若长时间未收到通知，应通过相应查询接口获取业务状态。
2. 需先解析通知内容中`msgtyp`字段的值获取通知子类型，各子类型如下：
    - NCCRTTRS：到账通知
    - NCDBTTRS：付款通知
3. `msgdat`为具体通知内容，字段说明如下：

### 通知内容字段说明
| 字段名称 | 字段ID | 数据类型 | 是否必输 | 描述 |
| ---- | ---- | ---- | ---- | ---- |
| 账号 | accnbr | string | - | - |
| 户名 | accnam | string | - | - |
| 交易日期 | trsdat | string | Y | 交易发生的日期 |
| 交易时间 | trstim | string | Y | 交易发生的时间，只有小时有效 |
| 币种 | c_ccynbr | string | - | - |
| 交易金额 | c_trsamt | string | Y | - |
| 收/付方帐户名称 | rpyacc | string(35) | N | 收/付方帐户名称 |
| 收/付方的转入或转出帐号 | rpynam | Z(62) | N | 收/付方的转入或转出帐号 |
| 帐户的联机余额 | blvamt | string | N | 帐户的联机余额 |
| 流水号 | refnbr | string(15) | Y | 银行会计系统交易流水号，可以和回单命名中的流水号关联 |
| 起息日 | vltdat | string(8) | N | 开始计息的日期 |
| 交易类型 | trscod | Z(12) | N | 见附录A.9 |
| 摘要 | naryur | Z(62) | N | 若为企业银行客户端经办的交易，则该字段为用途信息（4.0版代发代扣业务除外）；若为其它渠道经办的交易，则该字段为交易的简单说明和注解，显示在电子回单摘要上 |
| 借贷标记 | amtcdr | string(1) | Y | C:贷；D:借 |
| 流程实例号 | reqnbr | string(10) | N | 企业银行交易序号，唯一标示企业银行客户端发起的一笔交易 |
| 业务名称 | busnam | Z(32) | N | - |
| 用途 | nusage | Z(62) | N | - |
| 业务参考号 | yurref | string(30) | N | 企业银行客户端录入的业务参考号；用企业银行做的交易会有业务参考号，没有票据号；在柜台或其它地方生成的交易有票据号或其它唯一标识，统一称为业务参考号 |
| 业务摘要 | busnar | string(200) | N | 对业务的简单说明或注解，企业银行客户端录入的摘要信息 |
| 其它摘要 | otrnar | Z(62) | N | 对业务的其它说明或注解（暂不使用） |
| 收/付方开户地区分行号 | rpybbk | string(2) | N | 见A.1 招商分行，收/付方帐号开户行所在地区，如北京、上海、深圳等 |
| 收/付方开户行行号 | rpybbn | string(20) | N | 收/付方帐号的开户行的行号 |
| 收/付方开户行名 | rpybnk | Z(62) | N | 收/付方帐号的开户行的行名 |
| 收/付方开户行地址 | rpyadr | Z(62) | N | - |
| 母/子公司所在地区分行 | gsbbbk | string(2) | N | 见A.1 招商分行，母/子公司帐号的开户行所在地区，如北京、上海、深圳等 |
| 母/子公司帐号 | gsbacc | string(35) | N | - |
| 母/子公司名称 | gsbnam | Z(62) | N | - |
| 信息标志 | infflg | string(1) | N | 用于标识收/付方帐号和母/子公司的信息：为空表示付方帐号和子公司；为“1”表示收方帐号和子公司；为“2”表示收方帐号和母公司；为“3”表示原收方帐号和子公司 |
| 有否附件信息标志 | athflg | string(1) | N | Y：是；N：否 |
| 票据号 | chknbr | string(20) | N | - |
| 冲帐标志 | rsvflg | string(1) | N | *为冲帐，X为补帐（冲账交易与原交易借贷相反） |
| 扩展摘要 | narext | string(34) | N | 有效位数为16 |
| 交易分析码 | trsanl | string(6) | N | 1-2位取值含义见附录A.8交易分析码，3-6位取值含义见trscod字段说明 |
| 商务支付订单号 | refsub | string(50) | N | 由商务支付订单产生 |
| 企业识别码 | frmcod | string(10) | N | 开通收方识别功能的账户可以通过此码识别付款方 |
